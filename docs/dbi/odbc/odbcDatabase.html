void connect(char[] params, char[] username = null, char[] password = null);

Connect to a database using ODBC.

This function will connect DSN-lessly if params has a "=" and with DSN otherwise. For information on how to use connect DSN-lessly, see the ODBC documentation.

BUGS:
Connecting DSN-lessly ignores username and password.

Params:
char[] params The DSN to use or the connection parameters.
char[] username The username to connect with.
char[] password The password to connect with.

Throws:
DBIException if there was an error connecting.

Examples:

	OdbcDatabase db = new OdbcDatabase();
	db.connect("Data Source Name", "_username", "_password");

See Also:
The ODBC documentation included with the MDAC 2.8 SDK.
