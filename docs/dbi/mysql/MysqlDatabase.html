void connect(char[] params, char[] username = null, char[] password = null);

Connect to a database on a MySQL server.

Params:
char[] params A string in the form "keyword1=value1;keyword2=value2;etc."
char[] username The username to connect with.
char[] password The password to connect with.
Keywords: dbname = The name of the database to use.

host = The host name of the database to connect to.

port = The port number to connect to.

sock = The socket to connect to.

Throws:
DBIException if there was an error connecting.

ConvException if port is provided but is not an integer.

Examples:

	MysqlDatabase db = new MysqlDatabase();
	db.connect("host=localhost;dbname=test", "username", "password");
